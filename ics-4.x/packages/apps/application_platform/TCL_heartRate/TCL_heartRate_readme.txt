MS901-H6600平台

注：该程序需要用到SDcard去存储测试人员的图像；所以需要首先插入sd卡，最好是联网；
1 把haar_alt2 放在system/lib下面；
2 把libheartrate.so psuh到system/lib下面
3 把TCL_heartRate.apk 用adb install TCL_heartRate.apk 安装apk.

然后可以测试了。

测试过程：先检测人脸；然后15秒之后开始测试心跳。
