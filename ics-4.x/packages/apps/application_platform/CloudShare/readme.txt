 

1、  APK名称：TCL_common_CloudShare.apk

2、  应用英文名称：Cloud CShare

3、  应用中文名称：云分享

4、  包名：com.tcl.common.cshare

5、  主Activity： com.tcl.common.cshare/.CShareAccountListActivity

6、  本应用文件夹所有文件列表(apk,so,jar等)：
     TCL_common_CloudShare.apk:云分享的apk
  依赖的apk：
     ReconovaService.apk：人脸识别的service
    
  依赖的so库：
     libFaceProc_TCL_noNeon.so
     libImageUtil_TCL_noNeon.so
     libRecoMemoryFile.so

7、  本应用依赖文件：

     /system/lib/libFaceProc_TCL_noNeon.so
     /system/lib/libImageUtil_TCL_noNeon.so
     /system/lib/libRecoMemoryFile.so

8、  功能简介或说明
     云分享主要是基于电视为用户提供一个和好友交流互动的平台，在分享微博（新浪微博）的同时我们也可以将上传到我家空间中的文件分享微博话题中。

9、  集成方式：
    ReconovaService.apk是人脸识别的service
    TCL_common_CloudShare.apk是云分享的apk
    将这两个应用 push 到 system/app 下
