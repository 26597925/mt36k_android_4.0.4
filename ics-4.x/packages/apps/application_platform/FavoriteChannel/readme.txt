1.应用名称(launcher中显示的名字) ：偏爱频道

2.应用的包名：com.tcl.common.easyshow

3.主activity名：com.tcl.common.easyshow.HostActivity

4.相关文件说明(apk,so,jar等)：
TCL_Common_EasyShow.apk  偏爱频道apk
 


5.依赖的模块说明:
       5.1 依赖于tvservice,提供频道节目获取和跳转

6.集成的方法:
       将TCL_Common_EasyShow.apk拷贝到/system/app下。
注：如果安装apk文件，执行如下命令：
        adb install   /XXXX/XXX/TCL_Common_EasyShow.apk
        说明 ：/XXXX/XXX代表你存放以上2个文件的目录。

7.功能简介:
  能提供视频分类,直播,预约,收藏和点播等功能。
